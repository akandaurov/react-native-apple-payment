# react-native-apple-pay

Apple Pay implementation for React Native

## Installation

```sh
npm install react-native-apple-pay
```

## Usage

```js
import ApplePay from "react-native-apple-pay";

// ...

const result = await ApplePay.multiply(3, 7);
```
